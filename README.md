# America Gaona Borges 

Personal portfolio website showcasing my software engineering journey, projects, and experiences as a Computer Science student at Berea College.

## Features

- **Responsive Design**: Mobile-first approach using Bootstrap 5.3.2
- **Project Showcase**: Categorized projects with color-coded badges (Web Dev, UI/UX, ML/Robotics)
- **Interactive Components**: Smooth carousels, hover effects, and navigation
- **Professional Experience**: Detailed work history and technical skills
- **Leadership Section**: Community involvement and organizational roles
- **Accessibility**: Semantic HTML, descriptive alt text, and ARIA labels
- **Performance Optimized**: Lazy loading, SRI hashes, and modular CSS architecture

## Technologies Used

### Frontend
- **HTML5** - Semantic markup
- **CSS3** - Modern styling with Flexbox and Grid
- **JavaScript (ES6+)** - Interactive functionality
- **Bootstrap 5.3.2** - Responsive framework
- **Bootstrap Icons 1.11.3** - Icon library

### Key Features
- **Modular CSS Architecture** - Separated into base, main, and project-detail stylesheets
- **Lazy Loading** - Optimized image loading for performance
- **SRI Hashes** - Subresource Integrity for CDN security
- **SEO Optimized** - Meta tags, Open Graph, and semantic HTML

## Pages

### Home Page (index.html)
- **About**: Introduction with dynamic hero carousel
- **Experience**: Work history with expandable cards (2 internships)
- **Projects**: Portfolio of 6 projects with category badges
- **Leadership**: Community involvement and organizational roles
- **Contact**: Social links 

### Project Detail Page (project-detail-uiux.html)
- Comprehensive UI/UX case study for Berea CS Slack Optimization
- Research findings (8 layout images)
- Design iterations (3 initial designs)
- Final prototype demonstration
